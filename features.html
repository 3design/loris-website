Data Collection

LORIS makes it easy to manage and curate large amounts of data from a variety of sources and formats, with multiple options for input, identification and classification.

LORIS can accommodate any number of collection sites, complex rule-based visit schedules, project and subject cohort groups. With a variety of customizable fields for organization and sorting, researchers can access the information they need when they need it. Easy-to-use data entry interfaces and customizable settings to match the parameters of a study dataset, improve efficiency and accuracy in data collection.

  • Behavioural And Clinical Data

    LORIS allows you to enter behavioural and clinical data gathered from a variety of sources over time.

  • Imaging Data

    Compatible with many types of imaging device file formats (ie. MINC, DICOM, NiFtI) LORIS is designed for maximum compatibility with analyses with software packages like MINC, FSL, SPM, FreeSurfer.

  • Genetic Data

    The Genomic module enables exploration of genomic variant data (CNV, SNP) by sorting and filtering them according to a given annotation, subject attribute(s) and genomic location(s). A direct link to the UCSC Genome Browser allows access to genomic contextual information of individual variant locations.

  • Biosamples

    LORIS provides customizable modules for tracking biosample metadata including information on blood samples, cell culture, and environmental samples collected for genomic analysis.

Data Sharing

Powerful querying and dissemination tools let researchers process and distribute data in a flexible and efficient manner.

  • Data Query Tool

    The LORIS database handles large datasets efficiently, and empowers users with custom query building tools for generating custom curating datasets for analysis and dissemination. LORISâ€™s complete architecture includes NoSQL extension for rapid querying and retrieval capability on very large datasets.

  • File Downloading

    LORIS allows the user to package data for downloading to a local drive permitting data to be moved and saved.

  • Data Permissions

    LORIS can be customized with full user control, featuring multiple levels of permissions and roles, from superuser to radiologist to data entry staff. A combination of fully customizable permissions that are module, site, and edit/view specific provide full control over data entry, access and modification.

  • User Permissions

    LORIS can customize access privileges and restrict authorization to specific users or study groups.

Quality Control

LORIS makes it easy to manage large amounts of data. With a variety of customizable fields for organization and sorting, LORIS can help you access the information you need when you need it.

During data entry, strict completion rules, restricted field types and rule checking actively enforce data validity according to project standards.

  • Double Data Entry and Conflict Resolver

    LORIS can identify and highlight entry conflicts to assist the user in spotting errors in entered data.

  • Imaging Feedback

    Visual feedback allows the user to quickly see data relationships across the study.

  • Radiological Reviews

    LORISâ€™ Radiological Review Module allows radiologists to independently review candidate scans, leave feedback, and flag conflicts between reviews to include or exclude a given candidate.

  • Outlier Detection

    Our interactive statistics module allows for at a glance outlier detection, this notiffication can also appear on our dashboard notifying users upon log in of the statistical outlier.

  • PROTOCOL VIOLATION SCAN

    LORISâ€™ MRI Protocol Violation Module flags images that do not meet study protocol, with customizable criteria based on specific study and image requirements.

Neuroimaging

LORIS features several tools including MRI scan uploading, image pre-processing, and the integrated 3D Brain Browser to assist in understanding data and data results.

  • MRI Uploading

    LORIS allows the user to upload a variety of information collected from MRI scans.

  • Imaging Pre-Processing

    LORIS is designed to assist in understanding the results of data entered by performing pre-processing before imaging.

  • Multi-Modal

    LORIS can be customized to display data from various sources across timeframes set by the user.

  • Brain Browsing

    LORISâ€™ 3D Brain Browser is fully integrated to allow the analysis of data through direct comparison to a 3D-rendered model.

Visualization

LORIS includes several different data visualization toolsets, ranging from dashboard graphs to statistical plots and 3D imaging visualization, enabling users to animate and explore datasets in a variety of formats.

Data views and graphical display tools handle multiple formats and modalities.

  • Study Statistics

    A variety of tools allow the user to review and visualize summary metrics on data across all modalities and stages of data collection in LORIS.

Genomics

Tools for hosting, visualizing, and disseminating summary genetic and epigenetic data enable researchers to cross-link subject data across imaging, behavioural, and genetic elements of interest.

Summary genetic and epigenetic data are accessed via an embedded Genomic Browser module. The Genomic Browser enables cross-modal subject profiling, querying, visualization and data dissemination by providing links between subjectsâ€™ imaging, clinical and genetic information.

Genetic datasets (CNV, SNP, CpG, GWAS) are imported directly into LORIS from genetic analysis pipelines or other sharing platforms (e.g., CBRAIN).

Integrated links to external reference resources (such as the UCSC Genome Browser) facilitate rapid visualization of results.

Customization

OPEN SOURCE codebase and user-friendly Configuration module provide easy, transparent access for tailoring LORIS to any study, and extending it with custom features.

LORIS architecture includes configurable data collection design, adjustable layout features, and tools for building targeted data views according to study specifications.

  • MULTI-PURPOSE

    LORIS can be customized in a variety of ways to accommodate many kinds of study requirements, making LORIS one of the most versatile data management softwares available.

  • OPEN SOURCE

    LORIS is OPEN SOURCE, allowing full access and the capability to modify all aspects of their custom instance.

  • CONFIGURATION MODULE

    LORIS features and data parameters can be modified directly in the front-end Configuration Module, enabling easy customization and setup to meet study specifications.

  • EASILY ADD/REMOVE MODULES

    LORIS is designed to easily allow modules to be added and removed as needed..

MOBILE-FRIENDLY

LORIS uses responsive design technologies to ensure compatibility with mobile devices including tablets.

With the ability to customize the GUI as well as create your own custom CSS file, LORIS can be optimized to work on almost any device.
